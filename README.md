# LibraryProject

This is a simple library management system implemented in Python with a Tkinter-based graphical user interface.

## Features

- Display the library
- Add a book
- Update bookmark
- Mark as completed
- Delete a book
- Exit the program

### Prerequisites

- Python 3.x
